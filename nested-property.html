nested property adalah property yang berada didalam property
pada nested-property.css tiap property didefinisikan secara terpisah.
sehingga jika ada property didalam property, maka harus ditulis nama property induk dan nama propertynya.
seperti ".title .slogan". jika hanya sedikit property dan sedikit property didalam property maka tidak akan terasa.
namun, jika ada banyak property dan property didalam property, maka akan sangat panjang sekali untuk menuliskan tujuan propertynya.
jika, menggunakan scss, maka jika ada property didalam property, cukup untuk mendeklarasikan property didalam property induknya.
